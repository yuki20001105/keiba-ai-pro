- 旧モデル: テスト AUC 0.844 >> CV AUC 0.739（乖離 0.105）
- データ 4752 行 / 408 レースは少ない
- 時系列分割でテストセットが直近レースのみ → 季節性パターンを学習した可能性
- → 1年間データでの再評価が必要

### 3-4. speed_deviation の 2024-2026 小サンプル問題

- 1年間データ（約 2000 行想定）では過去と比較して不安定になる可能性
