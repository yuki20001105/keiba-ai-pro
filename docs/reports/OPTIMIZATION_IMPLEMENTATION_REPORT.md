## トラブルシューティング

### lxmlパッケージが見つからない場合
```bash
pip install lxml
```

### パース結果が異なる場合
lxmlは`html.parser`より厳密にHTMLを解釈します。
不正なHTMLの解釈が異なる可能性があります。

**対処法:**
```python
# 元に戻す場合
soup = BeautifulSoup(content, 'html.parser')
```

### メモリ使用量が増加した場合
lxmlはC実装のため、一般的にはメモリ効率が良いですが、
環境によっては増加する可能性があります。

**監視方法:**
```powershell
# タスクマネージャーでPythonプロセスを確認
# または
Get-Process python | Select-Object CPU,WorkingSet
```
